# 入门指南

根据你的开发场景选择合适的工具链：

- [Leo](/build/leo/getting-started/installation) - 使用 Leo 编写、编译和部署 Aleo 程序。Leo 是面向零知识应用的高级语言。
- [SDK](/build/sdk/getting-started) - 使用 Provable SDK 构建与 Aleo 网络交互的 JavaScript/TypeScript 应用。
- [Wallet Adapter](/build/wallets/wallet-adapter/getting-started) - 在 React 前端中接入 Aleo 钱包连接能力。
- [Embedded Wallets](/build/wallets/embedded-wallets) - 在前端中集成由 Dynamic 提供支持的嵌入式 Aleo 钱包。
